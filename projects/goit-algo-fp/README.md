# goit-algo-fp

Фінальний проєкт з курсу GoIT Algorithms.

## Опис проєкту

Проєкт містить реалізацію алгоритмів та структур даних відповідно до вимог фінального завдання:

* Завдання 1 — Однозв'язний список
* Завдання 2 — Дерево Піфагора (рекурсія)
* Завдання 3 — Алгоритм Дейкстри
* Завдання 4 — Візуалізація бінарної купи
* Завдання 5 — DFS та BFS
* Завдання 6 — Жадібний алгоритм та динамічне програмування
* Завдання 7 — Метод Монте-Карло

## Завдання 1. Робота з однозв'язним списком

Реалізовано:

* реверсування однозв'язного списку;
* сортування однозв'язного списку;
* об'єднання двох відсортованих однозв'язних списків в один відсортований список.

## Завдання 2. Рекурсія. Дерево Піфагора

Реалізовано рекурсивне побудування фрактала «Дерево Піфагора» за допомогою модуля `turtle`.

Можливості:

* візуалізація фрактала;
* введення користувачем рівня рекурсії.

## Завдання 3. Алгоритм Дейкстри

Реалізовано алгоритм Дейкстри для пошуку найкоротших шляхів у графі з використанням бінарної купи.

Функціональність:

* створення графа;
* пошук найкоротших шляхів від початкової вершини;
* використання пріоритетної черги для оптимізації обчислень.

## Завдання 4. Візуалізація бінарної купи

Реалізовано:

* побудову бінарної купи;
* графічну візуалізацію структури дерева.

## Завдання 5. Обхід дерева DFS та BFS

Реалізовано:

* обхід дерева в глибину (DFS);
* обхід дерева в ширину (BFS);
* використання стеку та черги;
* зміну кольорів вузлів відповідно до порядку обходу.

## Завдання 6. Жадібний алгоритм та динамічне програмування

Реалізовано два підходи до вибору набору страв при обмеженому бюджеті:

### Жадібний алгоритм

Вибір страв виконується за максимальним співвідношенням калорій до вартості.

### Динамічне програмування

Знаходиться оптимальний набір страв для отримання максимальної кількості калорій при заданому бюджеті.

## Завдання 7. Метод Монте-Карло

Реалізовано моделювання кидання двох гральних кубиків.

Виконано:

* велику кількість симуляцій;
* підрахунок частоти появи кожної суми;
* розрахунок емпіричних імовірностей;
* побудову таблиці та графіка результатів;
* порівняння отриманих результатів з аналітичними значеннями.

## Висновки

Результати, отримані методом Монте-Карло, наближаються до теоретичних імовірностей при збільшенні кількості симуляцій.

Найбільш імовірною сумою є 7, оскільки вона має найбільшу кількість можливих комбінацій випадання на двох кубиках.

Отримані результати підтверджують правильність роботи алгоритму та демонструють закон великих чисел: зі збільшенням кількості експериментів емпіричні значення наближаються до теоретичних.
