# Висновки

Було проведено порівняння алгоритмів пошуку підрядка: Кнута-Морріса-Пратта (KMP), Рабіна-Карпа та Боєра-Мура.

За результатами тестування алгоритм Боєра-Мура показав найкращу продуктивність для більшості випадків, особливо на великих текстах, завдяки механізму пропуску частин тексту.

Алгоритм KMP показав стабільну швидкість незалежно від структури тексту, але в середньому працював повільніше.

Рабін-Карп має хорошу швидкість, однак ефективність залежить від реалізації хешування.

Вбудовані механізми Python також демонструють високу швидкість завдяки внутрішнім оптимізаціям.
